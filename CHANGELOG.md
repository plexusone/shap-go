## [v0.4.0] - 2026-03-21

### Highlights

- DeepSHAP explainer - efficient SHAP values for neural networks using DeepLIFT
- GradientSHAP explainer - Expected Gradients method using numerical differentiation
- PartitionSHAP explainer - hierarchical Owen values for feature groupings
- AdditiveSHAP explainer - exact SHAP for Generalized Additive Models (GAMs)
- TreeSHAP interaction values - pairwise feature interaction computation
- CatBoost model support with JSON model parser
- ONNX-ML TreeEnsemble parser - load tree models directly from ONNX format
- Parallel batch explanation API with progress tracking
- Batched model predictions for efficient coalition evaluation
- Confidence intervals for sampling-based SHAP methods
- ONNX graph parsing and ActivationSession for intermediate layer access
