## [v0.2.0] - 2025-03-14

### Highlights

- TreeSHAP for XGBoost/LightGBM - exact SHAP values 40-100x faster than permutation methods
- LinearSHAP for linear models - O(d) closed-form SHAP computation
- Comprehensive MkDocs documentation site with guides and API reference

### Breaking

- `Masker.Mask()` now returns `([]float64, error)` instead of panicking on dimension mismatches ([`8f146ee`](https://github.com/plexusone/shap-go/commit/8f146ee))

### Upgrade Guide

- Add error handling to all `Mask()` and `MaskWithBackground()` calls
- Handle new sentinel errors: `ErrInstanceFeatureMismatch`, `ErrMaskFeatureMismatch`, `ErrBackgroundFeatureMismatch`
